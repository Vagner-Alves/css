Propriedade writing-mode

Define a orientação do texto ( vertical ou horizontal)

writing-mode: horizontal-tb

chega de saudade, sem ela não existe paz

writing-mode: vertical-rl

chega de saudade, sem ela não existe paz

writing-mode: vertical-lr

chega de saudade, sem ela não existe paz

writing-mode: sideways-lr

sou fã de maçãs, de todas as cores, vermelhas, verdes, amarelas e até azuis ( se elas existissem )

writing-mode: sideways-rl

sou fã de maçãs, de todas as cores, vermelhas, verdes, amarelas e até azuis ( se elas existissem )
