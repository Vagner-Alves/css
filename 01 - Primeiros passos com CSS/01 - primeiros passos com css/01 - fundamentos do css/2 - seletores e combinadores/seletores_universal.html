Seletores CSS

no css temos vários tipos de seletores

os seletores nos ajudam a implementar o código css na página

  • seletor por tipo: busca elementos por uma tag html específica
  • seletor por ID (#): busca elementos através do atríbuto "ID"
  • seletor por classe: busca elementos através do atríbuto "class"
  • seletor universal (*): seleciona todos os elementos HTML
  • seletor de atributo ([ atributo]): seleciona elementos que possuem um atributo específico no documento html e consegue também buscar atributos com valor específico
