Cores no CSS

existem várias formas de definirmos cores utilizando o CSS:

Cada cor possui 256 tonalidades ( 0 a 255 ) e abaixo estão algumas maneiras que definirmos essas cores:

  • Cores pré definidas
  • Palavra Chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com Transparencia
  • RGB
  • RGBA
  • HSL
  • HSLA

hs1 (hue, saturation, lightness)

  • Hue Matriz: é o grau na roda de cores ( 0 a 360) graus.
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation ( saturação): valor percentual
    • 0% um tom de cinza
    • 100% de cor total
  • Lightness ( luminosidade ): luminosidade da cor
    • 0% é preto
    • 100% é branco
