Proprieadades e Valores

Uma propriedade é uma característica de um elemento nosso HTML (cor de fundo, largura, altura, espaçamento, etc). Um valor define o resultado de uma propriedade e como o navegador deve exibir o estilo daquele elemento.

Formas de declarar o CSS

CSS inline: adicionamos o código css utilizando o atributo style dentro das tags HTML, elemento por elemento. 
 style="background-color: red;"
